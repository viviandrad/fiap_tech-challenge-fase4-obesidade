# Tech Challenge – Fase 4 – Data Analytics

Projeto desenvolvido como parte da Pós-Graduação em Data Analytics (FIAP).

## Objetivo
Desenvolver uma aplicação de Machine Learning para avaliação do risco de obesidade,
utilizando dados de hábitos alimentares, estilo de vida e indicadores físicos.

## Estrutura
- notebooks/: análise exploratória e modelagem
- models/: modelo treinado
- app/: aplicação Streamlit
- dashboard/: visualização de insights

## Tecnologias
- Python
- Pandas, Scikit-learn
- Streamlit

## Observação
A aplicação tem caráter de apoio à decisão e não substitui avaliação médica.
